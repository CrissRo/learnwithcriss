Lists

Unordered lists

Unordered lists are lists whose elements are marked with symbols.

Unordered list exemple:

< ul>

< li>Element 1< /li>

< li>Element2< /li>

< /ul>

  • Element1
  • Element2

< ul>< /ul>

Ul tag is used to define where the list start and end.

Ordered lists

Ordered lists are lists whose elements are marked with numbers.

Ordered list exemple:

< ol>

< li>Element 1< /li>

< li>Element2< /li>

< /ol>

  1. Element1
  2. Element2

< ol>< /ol>

Ol tag is used to define where the list start and end.

Type attribute

You can change the orther metods with type attributes.

Exemples:

type="1" (default)

type="A"    /"a" (arab letters)

type="I"/    "i" (roman numbers)
