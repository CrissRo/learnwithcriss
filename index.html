Introductions

What is HTML and where is used?

HTML is a markup language for documents designed to be displayed in a web browser and is frequently used in front-end development with CSS and JavaScript.

When HTLM was invented?

The first version of HTML was written by Tim Berners-Lee in 1993 and the latest version of HTML is HTML5.

How HTML work?

HTML documents consist of a series of elements, that elements tell browser how display the informations.
