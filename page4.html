Links

Basic links

< a href=" "> < a/>

This tag is used to add links on your website.

Between open tag and close tag you can write any message you want and this text will be transformed in a link.

For this tag will need href attribute.

Exemple:< a href="url_link">Text< /a>

Exemple 2:

This is a link to Google

< a href="https://www.google.com/">This is a link to Google< /a>

More about links

Target attribute

Exemple:

< a href="url link" target="value">Text< /a>

The "value" can be change with:

  1. _blank

    2. This value will open the linked document in a new window.

    Exemple link_Google
  2. _self

    3. This value will open the linked document on the same window (is default value).

    Exemple link_Google
  3. _top

Id attribute

The id attribute is used in links to redirectionate the user to a section from your website when the link is clicked.

To use an id attribute in a link need an id of an element.

To give an id to an element you need do that:

< h1 id= "element_id">Text< /h1>

Now the Text from h1 tag has the element id. To use this id in a link you need to do that:

< a href="#element_id"> Go to the element < /a>
