Basic Tags:

< !DOCTYPE >

DOCTYPE tag is used to tell the browser what version of html we use, for exemple if we use HTML 5 we write:

< !DOCTYPE HTML>

This usually is the first tag in our document.

< html> < /html>

Html tag is the container for all the others tags (< !DOCTYPE HTML> is an exception).

< head> < /head>

Head tag is used between html tag and body tag, in head tag we use other tags how don't change the aspect of the website.

< title> < /title>

Title tag is used inside the head tag and he gives the name of the website, this name is showed in the browser title bar.

< body> < /body>

Body tag is used after head tag. In body tag we use other tags as we can change the aspect of the website but we can also write text and this will showed in the website page.

< !-- -->

This tag is named comment tag and in this tag we can write notes about the code we write and this notes will not show in the website page.

Elements

In HTML we call an element everything from the start tag to the end tag.

Exemple:< tag_name> Text < /tag_name>

An element is formed from start tag, content and end tag.

Exemple:< tag_name> is the start tag, "Text" is the content and < /tag_name> is the end tag.

Some tags may not have an end tag or content

Attributes

In HTML attributes are additional informations for an element.

Attributes are always used in the start tag.

Exemple:< tag_name attribute_name="value"> Text < /tag_name>

Exemple2:< p title="a nice guy"> John < /p>

Title attribute add extra information about one element, in this case if we will put the cursor on word "John" we will see the attribut title, in this case "a nice guy".

Some more exemples of attributes are:herf,stye,scr,etc...

We will meet thats attributes again soon in the tutorial.
