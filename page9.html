5 more HTML tags

< source src="" type="">

The source tag is used to specify multiple media resources for media elements

With this tag you will use src attribute for source and type attribute for type of the file.

For exemple if ou have an mp4 vide you will have type="video/mp4".

< span> < /span>

Span tag is used to edit only a part form a text.

For exemple if you know css and want to use style attribute to style only one word in a paraghraph.

To do this you will open span tag before the word/words you want to style and colse after it.

< div> < /div>

Div tag is used to section a part of HTLM document.

Also is used as a container because is easy to style.

To section a part of a document you need introduce that part into a div tag.

< dl> < /dl>

Dl stand for description list and is a less used list in HTML.

Dl tag is used with dt tag, for define a term and dd tag to write definition of that term.

Dl is used same as ul and ol , but li is changed with dt and you have an extra tag, dd tag.

& nbsp;

Nbsp in not a tag but is very usefull and I think is worth to add in this list.

Nbsp is used to put space in a paragraph without convert it back.

Nbsp stand for non-breaking space

For exemple if you want have a pharagraf with an extra space you will write   ; where ou want an extra space.
