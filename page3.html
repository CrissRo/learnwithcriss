Pharagraphs

In HTML we can write text normal or we can use secific tags.

< p> < /p>

Pharagraph tag is used to convert normal text into a pharagraph.

Text inside the pharagraphs cannot have extra spaces or lines.

Exemple:

< p> Text   with       extra         spaces
and

extra

lines. < /p>

Will be converted in:

Text with extra spaces and extra lines.

< pre> < /pre>

Preformatted text tag is used to keep extra spaces and extra lines. Anyway will be differit.

< br>

Break tag is used to make a new line without start a new pharagraph.

< hr>

Horizontal rule tag is used to create a thematic break and separate the website's content.
