Meta tags

Meta tags are used to defines metadata about your website.

Also meta tags are used inside the head tag.

< meta charset="">

This meta tag is used to define charset.

Most people use UTF-8 charset.

< meta name="keywords" content="">

This meta tag is used to define keywords inside the content attribute.

For exemple for this website good keywords are "HTML ,tutorial, web".

< meta name="description" content="">

This meta tag is used to define description about the website.

For exemple for this website good description is "A tutrial site about how to make an website with HTML"

< meta name="author" content="John Doe">

This meta tag is used to define the author name.

For exemple author of this website is me, Criss.

< meta name="viewport" content="width=device-width, initial-scale=1.0">

This meta tag is used to make website look good on other devices like tablets and smartphones.
