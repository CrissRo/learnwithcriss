Tables

In HTML tables are used to arrange the information in rows and collons.

Exemple:

< table>


< tr>


< th>HEADER1< /th>

< th>HEADER2< /th>

< th>HEADER3< /th>


< /tr>

< tr>


< td>Data1< /td>

< td>Data2< /td>

< td>Data3< /td>


< /tr>

< tr>


< td>Data4< /td>

< td>Data5< /td>

< td>Data6< /td>


< /tr>


< /table>

HEADER1 HEADER2 HEADER3
Data1 Data2 Data3
Data4 Data5 Data6

< table>< /table>

Table tag is used to define where the table start and end.

< th>< /th>

Th tag is used to define a table header.

< tr>< /tr>

Tr tag is used to define a table row.

< td>< /td>

Td tag used to define a table data.

colspan="x"

Colspan attribute is used in a td or th tag and that data will have width of "x" colons.

Exemple:

HEADER1 HEADER2 HEADER3
Data1&2 Data3
Data4 Data5 Data6

Here first td tag have colspan="2".
